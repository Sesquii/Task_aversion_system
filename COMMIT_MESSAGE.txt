Purpose:
This change refines the graphic aid generation framework based on lessons learned
from the initial productivity score module implementation. The enhanced rules now
provide clear guidelines for:

1. Creating matching theoretical and data-driven charts with identical parameters
2. Generating 1-2 chart pairs per component factor
3. Creating comprehensive whole formula charts for score trends and correlations
4. Using consistent parameter ranges across all modules
5. Handling data extraction from both CSV and database formats

The productivity score module has been removed to allow regeneration using the
improved framework, ensuring better alignment between theoretical and data-driven
visualizations and more comprehensive whole formula analysis.

The rule enhancements will guide future module development and ensure consistent,
high-quality graphic aids across all analytics modules.
